Inter-FACE

Inter-FACE is an engaging and interactive web project designed to showcase creative use of HTML, CSS, and JavaScript. The centerpiece of this project is a playful digital "face" that responds dynamically to user interactions, making it a fun and visually appealing demonstration of front-end web design techniques.
